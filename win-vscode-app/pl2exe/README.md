# pl2exe README

はじめての拡張機能の作成テスト。現在vscode編集中の「***.pl」ファイルを、「***.exe」とする。
とはいっても、単純に c:/usr/perl/site/bin/ の中に個人的に入れている、pl2exe.plを実行しているだけ。

## Requirements

pl2exe.pl。引数に現在編集中の.plテキストのフルパスが渡る。

## Extension Settings

なし
