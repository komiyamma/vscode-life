「セットアップ」の注目点

  • .NET Framework 4.5.2

    VSCodeを動かすには、VSCode 4.5.2が必要。

  • コマンドでの起動

    カレントディレクトリで起動
    code .
    ファイルを指定して起動
    code ファイルパス
    ディレクトリを指定して起動
    code ディレクトリパス

    特にこの「ディレクトリを指定して起動」が便利。

    VSCodeは「プロジェクト」機能を持っており、
    Git/Githubや、拡張機能作成時など、
    「まとまったファイル群」として捉えることが出来る。

  • プロキシの設定

    自宅PCや自分のノートなどでは問題ないものの、
    会社など、イントラネットがプロキシを利用するPCでは、
    拡張モジュール等で失敗する場合がある。

    こういった環境ではプロキシを指定する。

    環境変数に、

    PCの環境変数の一例
    http_proxy=http://10.203.0.1:5187/

    などと、指定する。
    (IPアドレスやPort番号は、会社のプロキシのものを調べて指定する)
